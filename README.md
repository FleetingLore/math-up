# 环境搭建

克隆本存储库.

找我要文档生成工具.

学一下怎么用git.

自己创建存储库,然后连接到git,可以设置ssh.

# 构建

参考static目录去组织文档.入口是 `local.lore`.

有个适用于 intellij 的 lore 语言支持插件,可以找我要,或者自己去这个号的存储库列表找.

GitHub Pages 的网页显示目录设置成 `\docs`.

`\docs` 目录下的 `style.css` 可以自主修改.

用 python 安装 mkdocs.

# 布置

每次 commit 之前记得执行 `./wein`

push 的时候自动构建文档.
